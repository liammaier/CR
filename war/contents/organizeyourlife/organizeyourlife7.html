Showing/Hiding Calendars

The fact that you can view multiple calendars was mentioned previously. It’s time to learn how to choose the calendar(s) to be visible on the Calendar screen. Remember, if you only have one calendar, this doesn’t apply to you.

As referenced previously, the default setting for the Calendar app is to show All Calendars, as shown in Figure 7-1. This means, regardless of how many separate calendars (e.g., Home, Work, Kids, etc.) have been created, all appointments associated with every calendar are visible.

You are not stuck with the default option to show all calendars; you can select only certain calendars to be visible (or you can technically hide all calendars, but why would you do that?).

Figure 7-2 shows a sample default Calendars screen. Your screen will look different from the one below because I have several calendars set up in my iCloud account. A check to the right of the calendar name indicates that that calendar’s appointments will appear in the Calendar app. In contrast, Figure 7-3 shows the same Calendars screen with only two calendars checked (i.e., Home and ID4theWeb) to appear in the Calendar.

Showing and hiding calendars can be a tricky thing. It isn’t uncommon for someone to hide a calendar thinking that they will go back in and show it again when they are through. Unfortunately, the “going back” part often gets lost in the chaos of daily life. This can lead to many problems, including missed appointments, double-booking your time, etc.

Imagine the following scenarios:

  • You know you entered an appointment, but it’s just not showing up because that Calendar isn’t visible.
  • You schedule a new appointment not realizing that it conflicts with another appointment because all your calendars aren’t visible at the time.
  • You don’t pick up your kids because that calendar wasn’t visible.

The scenarios could go on and on, but I think you get the point. Showing and hiding calendars is a nice feature, but not one to be used without great caution and careful attention to detail. Establishing various calendars and assigning appointments to appropriate calendars is a great way to organize your busy life. Setting up different calendars requires planning and categorization skills. In addition, assigning each appointment to an appropriate calendar requires decision-making skills. It will be obvious what calendars are appropriate to assign to some events; however, there may be some activities that aren’t so obvious; this is where decision-making skills come into play.

Finally, problem-solving becomes a factor when a calendar is incorrectly assigned to an event. Recognizing the problem in the calendar assignment (e.g., a workout was assigned to my work calendar), identifying what the problem is (e.g., looking for my last workout appointment and not finding it in my fitness calendar), and resolving the problem (e.g., changing the assigned calendar event to the fitness calendar) are all part of the problem-solving process.

New Invitation Alerts

This option works with the Invitation Alerts previously discussed. The default setting is to receive invitation alerts; however, you can turn that feature off. The New Invitation Alerts option provides the option to turn alerts ON or OFF.

To Modify New Invitation Alerts

  • Tap the New Invitation Alerts option to toggle between ON and OFF.

Sync

The Sync option allows you to choose how far back in time you would like to synchronize events with another device’s calendar or with a cloud-based (online) calendar such as iCloud or Google. The options are:

  • Events 2 Weeks Back
  • Events 1 Month Back (default setting)
  • Events 3 Months Back
  • Events 6 Months Back
  • All Events
129Chapter 7: More About the Calendar App

It might seem unnecessary to sync events more than one month back; however, if your purpose is to use the device as a memory prosthetic, you will want to choose Sync All Events. You might be asking, “But, why?” The answer is that if you only sync events from one month back, any event prior to one month will not appear on your device. It will still appear in your iCal, iCloud, and/or Google Calendars, etc., but it won’t appear on your device. Ask yourself this question, “Which device will I tend to have available at any time or place if I need to see when was that last time I went to the neurologist?”

To Change the Sync Setting

  • Tap the Sync option.
  • Select the desired sync option (All Events is strongly recommended).
  • Tap the Mail… icon in the upper left corner to return to the Mail, Contacts, Calendar screen.
